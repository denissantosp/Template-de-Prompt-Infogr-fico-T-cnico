# 📐 Template de Prompt — Infográfico Técnico "Estilo Enciclopédia de Engenharia"

> Prompt **genérico e reutilizável** para gerar infográficos técnicos premium de qualquer produto, máquina, dispositivo ou sistema de interesse. Compatível com ChatGPT (DALL·E), Gemini (Imagen), Midjourney, Ideogram e outras IAs de geração de imagem.

## 📋 Sobre este prompt

Este template cria infográficos no estilo **enciclopédia de engenharia editorial**: layout limpo (Swiss design), tipografia grande e legível, paleta de cores sóbria, e foco em "conhecimento antes de densidade visual". Funciona para **qualquer tema técnico** — eletrônicos, veículos, máquinas industriais, instrumentos, arquitetura, etc. Basta preencher os campos `[ENTRE COLCHETES]`.

## ⚙️ Compatibilidade

| Ferramenta | Funciona? | Observações |
|---|---|---|
| **ChatGPT / DALL·E** | ✅ | Pode simplificar texto longo; envie em partes se necessário |
| **Gemini / Imagen** | ✅ | Bom para composições limpas estilo Swiss design |
| **Midjourney** | ⚠️ Parcial | Requer adaptação para parâmetros (`--ar 1:1`, `--style raw`) |
| **Leonardo / Ideogram** | ✅ | Ideogram tem bom desempenho com texto renderizado |

> ⚠️ **Nota sobre marcas registradas:** alguns geradores de imagem evitam reproduzir logotipos exatos de marcas comerciais (Apple, Cisco, Sony, etc.). Se a IA recusar ou gerar uma versão genérica, descreva o produto de forma funcional (ex.: "telefone IP corporativo empresarial" em vez do nome comercial exato) ou use o resultado apenas para fins educacionais, deixando claro que é uma representação não-oficial.

## 🎯 Prompt completo (template)

```text
Create a premium square "reference-style engineering infographic" centered
around [PRODUTO/TEMA DE INTERESSE], designed as a beautifully curated
technical handbook page rather than a commercial poster. The composition
should resemble a modern engineering encyclopedia, combining high-end
industrial design documentation, museum-quality editorial infographics,
and professional technical visualization systems.

VISUAL DIRECTION
• 1:1 square composition
• Large readable typography with strong editorial hierarchy
• ALL TEXT MUST BE LARGE, SHARP, HIGH-CONTRAST, AND EASY TO READ
• Oversized titles and medium-to-large paragraph text
• Avoid tiny fonts and micro-labels
• Reduce excessive information density
• Use fewer content blocks with larger spacing
• Clean Swiss-style editorial layout with breathable spacing
• Soft light-toned technical background
• Subtle blueprint textures and schematic overlays
• Elegant muted palette: graphite gray, titanium silver, deep navy,
  steel blue, soft white accents
• Rounded modular information cards
• Premium UI dividers and separators
• Gentle realistic shadows
• Minimal technical iconography
• Thin precision annotation lines
• Organized "knowledge-first" composition

MAIN SUBJECT PRESENTATION
Place an ultra-detailed realistic render of [PRODUTO/TEMA DE INTERESSE] at
the center in dramatic three-quarter perspective. The subject must feature:
• realistic industrial materials
• accurate reflections
• authentic textures appropriate to the subject
  (plastic, metal, fabric, glass, etc.)
• detailed [COMPONENTES VISÍVEIS RELEVANTES — ex: painel, tela,
  engrenagens, controles, estrutura]
• realistic lighting and shadows
• premium industrial realism
• ultra-sharp product detailing

TECHNICAL CALLOUTS
Surround the subject with LARGE READABLE engineering callouts explaining:
• [CARACTERÍSTICA TÉCNICA 1]
• [CARACTERÍSTICA TÉCNICA 2]
• [CARACTERÍSTICA TÉCNICA 3]
• [CARACTERÍSTICA TÉCNICA 4]
• [CARACTERÍSTICA TÉCNICA 5]
• [CARACTERÍSTICA TÉCNICA 6]
(adicione de 6 a 12 itens — funções, protocolos, materiais, mecanismos,
sistemas internos, etc., relevantes ao tema escolhido)

INFOGRAPHIC MODULES
Include premium editorial sections in PORTUGUESE (BRAZIL):
• Visão Geral do [Produto/Tema]
• Especificações Técnicas
• Arquitetura / Sistema Principal
• Segurança e Conectividade (ou "Segurança e Confiabilidade")
• Interface e Usabilidade
• Eficiência Energética / Desempenho
• Estrutura Interna
• Fluxo Operacional
• Linha do Tempo Tecnológica / Histórica
• Top 5 Curiosidades Técnicas
• Aplicações Práticas
• Legado e Importância no Setor

VISUALIZATION MODULES
Add small premium technical visualization components:
• performance charts
• flow diagrams
• miniature blueprint side views
• cutaway/exploded view diagrams
• system workflow graphics
• comparison or timeline graphics (se aplicável)

TYPOGRAPHY RULES
• Use LARGE HIGH-CONTRAST fonts
• Ensure all text remains readable even at thumbnail size
• Avoid condensed typography or tiny technical paragraphs
• Maintain generous spacing between modules
• Prioritize readability over information quantity

STYLE KEYWORDS
"premium engineering encyclopedia", "editorial industrial handbook",
"museum-quality technical infographic", "Swiss editorial design",
"ultra-detailed engineering visualization"

AVOID
tiny unreadable fonts, cluttered layouts, excessive information density,
cinematic action poster aesthetics, grunge textures, propaganda styling,
commercial advertising composition, overcomplicated micro-details

FINAL RESULT
The final artwork should resemble a professionally published engineering
reference-book page created for technology enthusiasts, collectors,
designers, engineers, and educational archives. All text must be written
in Portuguese (Brazil) with professional technical terminology and highly
readable typography.
```

## ✏️ Como personalizar (passo a passo)

| Campo no template | O que colocar |
|---|---|
| `[PRODUTO/TEMA DE INTERESSE]` | Nome do item — equipamento, veículo, máquina, sistema, conceito técnico |
| `[COMPONENTES VISÍVEIS RELEVANTES]` | Partes físicas visíveis e marcantes do objeto (tela, painel, motor, rodas, etc.) |
| `[CARACTERÍSTICA TÉCNICA 1–12]` | As funções, protocolos, materiais e sistemas mais importantes do tema |
| Seções dos módulos | Já são genéricas — ajuste só os nomes entre parênteses se não se aplicarem |

### 🔧 Exemplos rápidos de preenchimento

**Smartphone:**
`[PRODUTO/TEMA]` → "a flagship smartphone with foldable AMOLED display"
`[CARACTERÍSTICAS]` → câmera multi-lente, processador, bateria, 5G, biometria, resistência à água

**Carro elétrico:**
`[PRODUTO/TEMA]` → "a modern electric vehicle (EV) sedan"
`[CARACTERÍSTICAS]` → bateria de íon-lítio, motor elétrico, autonomia, frenagem regenerativa, sistema ADAS

**Motor de combustão:**
`[PRODUTO/TEMA]` → "a V8 internal combustion engine"
`[CARACTERÍSTICAS]` → sistema de injeção, ciclo de combustão, refrigeração, pistões, taxa de compressão

**Drone:**
`[PRODUTO/TEMA]` → "a professional quadcopter drone"
`[CARACTERÍSTICAS]` → estabilização por giroscópio, câmera gimbal, motores brushless, GPS, autonomia de voo

**Relógio mecânico:**
`[PRODUTO/TEMA]` → "a mechanical automatic wristwatch"
`[CARACTERÍSTICAS]` → movimento automático, escape, reserva de marcha, resistência à água, complicações

## 💡 Dicas de uso

- **Divida o prompt** em ChatGPT se a interface cortar texto longo — envie primeiro "VISUAL DIRECTION" + "MAIN SUBJECT", depois complemente com os módulos.
- Gere as instruções de estilo em **inglês** (a IA segue com mais precisão) e mantenha os títulos das seções em português, como já estruturado acima.
- Se o texto sair distorcido (comum em geração de imagem com texto), peça uma segunda iteração: "redo only the typography blocks, keep composition identical".
- Para temas com marca registrada, prefira descrever a categoria/função do produto em vez do nome comercial, evitando recusas do gerador.
